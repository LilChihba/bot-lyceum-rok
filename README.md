# Бот для лицея в игре "Rise of Kingdoms"
## Что делает данный бот?
Данный бот с легкостью скачивает с интернета вопросы лицея, сохраняет их в json файле, а после начинает считывать данные с экрана. После того, как бот считал данные с экрана он ищет в файле json неточные совпадения с вопросами и выдает на экран ответ на данный вопрос.
Бот предназначен для работы с экраном 1920x1080 в полноэкранном режиме.

## Перед стартом необходимо:

Проверить, установлен ли у вас python версии 3.10.5 или выше:
> python

Проверить, установлен ли у вас pip версии 22.1.2 или выше:
> pip --version

После, установить все необходимые библиотеки:
> pip install -r requirements.txt

Также для работы необходимо скачать и установить в папку с ботом Tesseract-OCR:
> https://github.com/UB-Mannheim/tesseract/wiki
